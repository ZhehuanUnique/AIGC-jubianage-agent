## 系统概述

**剧变时代Agent** 是一款专业的AI视频生成工具，支持从剧本输入到视频导出的完整工作流程。系统集成了多种AI模型，包括剧本分析、智能切分、文生图、图生视频等功能，帮助用户高效制作商业化级别的视频内容。

### 核心特性

- 📝 **智能剧本分析**：自动提取角色、场景、物品
- ✂️ **智能剧本切分**：自动将剧本切分为多个片段
- 🎨 **多模型文生图**：支持 Nano Banana Pro、Midjourney v7 等
- 🎬 **多模型图生视频**：集成豆包 Seedance、Google Veo3.1、Vidu V2、MiniMax Hailuo 等
- 🎥 **首尾帧生视频**：支持首帧+尾帧+提示词生成视频
- 🎤 **语音生成**：支持多种语音模型和参数调节
- 📊 **数据分析**：项目数据统计和分析
- 💰 **积分系统**：灵活的积分充值和消耗机制

## 快速开始

### 1. 登录系统

访问系统首页，点击"剧变时代Agent"按钮，使用您的账号登录。

### 2. 创建项目

- 在"项目管理"页面，点击"新建场景"按钮
- 输入项目名称和描述
- 点击"创建"完成项目创建

### 3. 输入剧本

- 进入项目后，在"片段管理"页面可以输入剧本
- 支持两种方式：
  - **文本输入**：直接在文本框中输入剧本内容
  - **文件上传**：上传 `.docx` 格式的剧本文件

### 4. 开始创作

按照系统提示，依次完成：
1. 剧本输入 → 2. 资产详情 → 3. 分镜管理 → 4. 融图管理 → 5. 视频编辑

## 核心功能

### 1. 项目管理

**功能位置**：首页 → 项目管理

**主要功能**：
- 创建新项目
- 查看项目列表
- 删除项目
- 项目详情查看

**操作步骤**：
1. 点击"新建场景"创建项目
2. 在项目列表中点击项目名称进入项目详情
3. 使用侧边栏导航访问项目的各个功能模块

### 2. 片段管理

**功能位置**：项目 → 片段管理

**主要功能**：
- 查看和管理剧本片段
- 每个片段对应一个分镜
- 支持片段删除（删除后积分不返还）

**注意事项**：
- 删除片段前会弹出确认对话框
- 删除操作不可恢复

### 3. 角色管理

**功能位置**：项目 → 角色管理

**主要功能**：
- 管理项目中的角色资产
- 上传角色图片
- 编辑角色信息
- 删除角色

### 4. 场景管理

**功能位置**：项目 → 场景管理

**主要功能**：
- 管理项目中的场景资产
- 上传场景图片
- 编辑场景信息
- 删除场景

### 5. 物品管理

**功能位置**：项目 → 物品管理

**主要功能**：
- 管理项目中的物品资产
- 上传物品图片
- 编辑物品信息
- 删除物品

### 6. 分镜管理

**功能位置**：项目 → 分镜管理（通过片段管理页面进入）

**主要功能**：
- 为每个分镜生成图片
- 配置分镜的融图提示词
- 选择文生图模型（Nano Banana Pro、Midjourney v7）
- 批量生成分镜图片
- 图片 Upscale（放大）

**操作流程**：
1. 系统自动为每个片段生成分镜提示词
2. 选择文生图模型和参数
3. 点击"生成"按钮生成分镜图片
4. 可选择对图片进行 Upscale 处理

### 7. 融图管理

**功能位置**：项目 → 融图管理（通过分镜管理页面进入）

**主要功能**：
- 选择分镜图片作为视频素材
- 配置视频生成参数：
  - 选择视频生成模型
  - 设置分辨率（480p/720p/1080p）
  - 设置视频时长（3-10秒）
  - 输入视频生成提示词
- 批量提交视频生成任务
- 查看视频生成进度

**支持的视频生成模型**：
- 即梦AI-视频生成3.5pro（doubao-seedance-1-5-pro-251215）
- ViduQ2-Turbo
- Google Veo3.1
- Google Veo3.1-Pro
- MiniMax Hailuo-02
- MiniMax Hailuo-2.3
- MiniMax Hailuo-2.3-fast

### 8. 视频编辑

**功能位置**：项目 → 视频编辑（通过融图管理页面进入）

**主要功能**：
- 查看所有生成的视频
- 视频预览和播放
- 视频下载
- 视频参数调整和重新生成
- 视频审核

### 9. 首尾帧生视频

**功能位置**：项目 → 片段管理 → 首尾帧生视频

**主要功能**：
- 上传首帧图片（必需）
- 上传尾帧图片（可选）
- 输入视频生成提示词
- 选择视频参数（分辨率、时长）

**使用场景**：
- **单首帧模式**：仅上传首帧 + 提示词，生成视频
- **首尾帧模式**：上传首帧 + 尾帧 + 提示词，生成视频（仅支持3.5pro模型）

- 尾帧为可选，如果上传尾帧，系统会自动使用3.5pro模型

## 功能详解

### 剧本分析和切分

系统使用通义千问大模型进行剧本分析和切分：

**剧本分析**：
- 自动提取角色、场景、物品
- 识别剧本中的关键信息

**剧本切分**：
- 智能将剧本切分为多个片段
- 每个片段对应一个分镜
- 所有片段合起来等于完整剧本，不遗漏任何内容
- 根据剧本的自然段落、场景转换、对话切换等逻辑进行切分
- 每个片段适合制作5-10秒的视频分镜

**分镜提示词生成**：
- 为每个分镜自动生成详细的中文文生图融图提示词
- 提示词格式：`景别：主体: 风格: 构图: 氛围：`

### 文生图功能

**Nano Banana Pro**：
- **特点**：角色一致性、高级文本渲染、4K输出
- **适合**：需要角色一致性的连续剧集
- **分辨率**：1K、2K、4K
- **宽高比**：auto、16:9、1:1、9:16、21:9
- **价格**：¥0.09~¥0.18/次（1800积分/次）
- **获取API Key**：https://grsai.com/zh/dashboard/documents/nano-banana

**Midjourney v7 t2i**：
- **特点**：高分辨率、多样化风格、出色的光影表现
- **适合**：高质量艺术创作、风格化图像
- **分辨率**：最高 4096×4096 像素
- **宽高比**：通过提示词参数控制（--ar 16:9等）
- **价格**：$0.025/次
- **获取API Key**：https://302.ai

### 图生视频功能

系统支持多种图生视频模型，每个模型有不同的特点和适用场景。

## 积分系统

### 积分获取

- 通过"积分充值"页面购买积分
- 支持多种充值套餐
- 1元人民币对应5个积分

### 积分消耗

- 文生图：根据模型不同，消耗不同积分
- 图生视频：根据模型、分辨率、时长不同，消耗不同积分
- 各项目积分消耗在提交任务处有醒目标注

### 积分返还

- 因服务器原因生成失败的返还积分
- 删除片段不会返还积分

### 积分有效期

- 购买后的积分有效期为365天
- 到期余量自动清零

## 常见问题

### 1. 如何创建新项目？

在"项目管理"页面，点击"新建场景"按钮，输入项目名称和描述，点击"创建"即可。

### 2. 如何上传剧本？

在项目的"片段管理"页面，可以直接在文本框中输入剧本，或者上传`.docx`格式的剧本文件。

### 3. 如何生成分镜图片？

在"分镜管理"页面，选择文生图模型和参数，点击"生成"按钮即可。支持批量生成。

### 4. 如何生成视频？

在"融图管理"页面，选择分镜图片，配置视频生成参数（模型、分辨率、时长、提示词），点击"提交任务"即可。

### 5. 首尾帧生视频如何使用？

在"片段管理"页面，点击"首尾帧生视频"按钮，上传首帧图片（必需）和尾帧图片（可选），输入提示词，选择参数，点击"生成视频"即可。

### 6. 如何充值积分？

点击导航栏的"积分充值"按钮，选择充值套餐，完成支付即可。

### 7. 积分如何消耗？

在提交任务时，系统会显示本次任务消耗的积分数量。不同模型和参数消耗的积分不同。

### 8. 视频生成失败怎么办？

如果因服务器原因导致视频生成失败，系统会自动返还积分。可以在"视频编辑"页面查看失败原因。

### 9. 如何删除项目？

在"项目管理"页面，找到要删除的项目，点击删除按钮，确认删除即可。注意：删除项目会同时删除项目中的所有数据，包括片段、分镜、视频等。

### 10. 如何退出登录？

将鼠标移动到导航栏右上角的用户信息区域，会显示红色的"退出登录"文字，点击即可退出登录。
